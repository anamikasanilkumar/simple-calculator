🧮 Simple Calculator

A lightweight and responsive calculator web application built using HTML, CSS, and JavaScript. It performs basic arithmetic operations with a clean and modern user interface.

🚀 Features

• Perform basic calculations: Addition, Subtraction, Multiplication, Division
• Clean and minimal UI design
• Responsive layout using Flexbox and CSS Grid
• Instant input display
• Clear ("C") button to reset calculations
• Handles invalid inputs with error display

---

🛠️ Built With

• HTML
• CSS
• JavaScript 


⚙️ How It Works

• User inputs are displayed dynamically on the screen
• JavaScript functions handle input, clearing, and calculation
• The "eval()" function is used to compute the result of the expression

---

⚠️ Disclaimer

This project uses JavaScript's "eval()" function for calculation. While suitable for simple projects, it is not recommended for production use due to security concerns.

---

💻 Usage

1. Clone or download the repository
2. Open "index.html" in any modern web browser
3. Start calculating
